## ✍️ 正文文案

**标题（18 字内，带关键词）：**
> 给我的 Windows 装了个灵动岛，同事以为我换了 Mac

**正文：**

```
家人们谁懂啊！😭
Windows 桌面终于也能拥有灵动岛了！

这个小工具叫 MyIsland，开源免费，下载 exe 直接运行就行，不用安装。

🎵 听歌的时候顶部会浮出一个小药丸，点一下展开就是实时歌词，还有专辑封面和音频可视化，氛围感直接拉满。

💧 最让我惊喜的是喝水提醒！设置好时间，到点了它会整岛弹窗提醒我喝水，打工人的养生福音。

🎨 有三种皮肤可以换：默认、Win11 Mica、动态取色（会根据专辑封面自动变颜色），每个都很好看。

🔧 双击展开/收起，滚轮切换页面，右键托盘就能进设置，上手零门槛。

关键是它用 Rust 写的，动画特别丝滑，一点都不卡。

GitHub 搜 xiaoxiaobaijiangjun/MyIsland 就能找到，觉得好用的朋友可以顺手点个 ⭐ 支持下开发者～

#Windows美化 #桌面美化 #灵动岛 #电脑桌面 #效率工具 #开源 #数码好物 #晒晒我的桌面 #MyIsland #windows11
```
