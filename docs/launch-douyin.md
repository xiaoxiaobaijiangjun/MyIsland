## ⚠️ 抖音发布注意事项

1. **不要直接写完整 GitHub 链接在标题/正文**，容易被限流；放在评论区置顶更安全。
2. 视频里出现 GitHub 地址时，**用录屏框住链接**，让观众自己打或复制。
3. 发布时定位选「数码」或「科技」类，算法更容易推给目标人群。
4. 前 3 秒决定完播率，务必先把"Windows 灵动岛"这个点抛出来。
